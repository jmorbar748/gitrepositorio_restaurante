## Descripción del Proyecto

El objetivo es crear una página web sencilla para un restaurante, que incluya:
*   Una página de inicio (`index.html`).
*   Una página de menú (`menu.html`) con platos típicos andaluces.
*   Un formulario de contacto (`contacto.html`).

## Tecnologías Utilizadas

*   **HTML5**: Para la estructura de las páginas.
*   **CSS3**: Para los estilos y el diseño responsive.
*   **Git**: Para el control de versiones.
*   **GitHub**: Como repositorio remoto.

## Características

*   Diseño limpio y adaptado a móviles (responsive).
*   Menú interactivo con platos de la gastronomía andaluza.
*   Formulario de contacto funcional.
